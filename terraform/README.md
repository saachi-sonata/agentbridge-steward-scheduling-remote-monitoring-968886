# AgentBridge Blueprint: Steward • Scheduling (Remote Monitoring)

> Deployed on **Azure Virtual Machine** via Terraform IaC

## Quick Start

```bash
# 1. Copy and fill in your credentials
cp terraform.tfvars.example terraform.tfvars

# 2. Initialize Terraform
terraform init

# 3. Preview changes
terraform plan

# 4. Deploy
terraform apply
```

## Architecture

This deployment uses AgentBridge's 6-layer agentic architecture:

| Layer | Technology | Purpose |
|-------|-----------|---------|
| Reasoning | gpt-4o | Decision-making and inference |
| Orchestration | n8n | Workflow execution and coordination |
| Protocol | MCP | Inter-agent communication |
| Governance | OPA | Policy enforcement and compliance |
| Observability | Prometheus | Monitoring and tracing |
| Intelligence | vector-db | Knowledge retrieval and memory |

After `terraform apply`, you'll see:
- **Public IP** — Instance address
- **SSH Command** — Direct SSH access
- **Application URL** — Workflow editor at port 5678
- **Health URL** — Health check endpoint
- **Console URL** — Cloud provider console link
